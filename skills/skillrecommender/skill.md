## Scoring System

**Scoring:** Keywords (3pts) + Task alignment (3pts) + Domain (2pts) + Complexity (2pts)

**Recommendations:**
1. Match keywords from user request
2. Assess task complexity (simple vs complex)
3. Identify domain (UI, API, Testing, etc.)
4. Return top 3-5 skills ranked by score
