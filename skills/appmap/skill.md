# AppMap Skill

Generate architecture artifacts. **Triggers:** `AppMap`, `generate sitemap`, `map the app`
**Output:** `APP_MAP.md`

**4 Artifacts:**
1. **Sitemap:** Route hierarchy, auth, dynamic params
2. **Workflow:** User flows, state transitions, API sequences
3. **Component Directory:** Pages, Components, Hooks, Services, Contexts
4. **UI/UX Diagram:** ASCII layouts, component trees, interactive elements

**Execution:** 4 Opus agents parallel → Synthesize → Validate
