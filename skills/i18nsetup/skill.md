# i18nSetup Skill

Internationalization. **Triggers:** `i18nSetup`, `setup i18n`, `translations`

| Library | Framework |
|---------|-----------|
| next-intl | Next.js |
| react-i18next | React |
| lingui | Any (compile-time) |

**Setup:** locales config, middleware, message files (JSON with ICU)

**Best Practices:** Namespace keys (`auth.login`), Named params (`{name}`), Intl formatters, RTL support
