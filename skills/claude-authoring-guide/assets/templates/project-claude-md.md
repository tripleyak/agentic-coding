## Template Usage Instructions (delete after customizing)

### Purpose
This template creates a project-level CLAUDE.md file that:
- Provides project-specific context
- Defines local development standards
- Loads relevant skills automatically
- Documents domain knowledge

### File Location
Save to: `{{project-root}}/.claude/CLAUDE.md`

### What to Include

**Always include:**
- Tech stack and key directories
- Verification commands specific to project
- Project-specific coding standards

**Include when relevant:**
- Domain knowledge glossary
- Architecture decisions
- Known constraints or gotchas
- Project-specific skills to load

**DO NOT include:**
- Global standards (already in ~/.claude/CLAUDE.md)
- Generic best practices
- Framework documentation

### Quality Checklist
- [ ] Specific to this project (not generic)
- [ ] Verification commands work
- [ ] Domain terms defined
- [ ] No placeholder text remaining
- [ ] Complements (not duplicates) global CLAUDE.md
